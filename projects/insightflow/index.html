AI WORKFLOW / LOCAL TOOL

AI InsightFlow

一个面向发布会和长视频内容的 AI 工作流工具：上传视频后自动提取关键画面、转写音频内容，经过人工确认后生成结构化洞察报告，并导出 Word 文件。

TOOL ENTRY

同仓库工具入口

工具代码已经放进个站仓库的 apps/insightflow。本地预览时先启动 Streamlit，再从这里进入工具。

打开工具 运行说明
01 Upload Video
02 Extract Materials
03 Confirm Slides & Text
04 Generate Report
05 Export Word

为什么不能只复制成 HTML？

InsightFlow 需要处理视频、抽帧、提取音频、转写和生成报告，这些都需要 Python 后端能力，纯静态页面无法完成。

现在怎么和个站放一起？

个站负责项目入口和展示，工具源码放在同一个仓库中。部署时只要选择支持 Python 的服务，就能跟个站一起维护和上线。

后续上线怎么处理？

上线后把按钮里的本地地址替换成部署后的 InsightFlow 地址。如果部署平台支持同域服务，也可以挂到同一个域名下。

Local source: /apps/insightflow · Local app: http://localhost:8501 · Replace with production URL when deployed.
